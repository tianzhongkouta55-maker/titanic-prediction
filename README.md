<summary> 1. 学習内容と習得した技術 </summary>

#### 🛠 トラブルシューティング
- **FileNotFoundError への対応**: 
  プログラムがデータファイル（train.csv）を参照できないエラーに直面し、パスの指定、または外部URLからの直接読み込みによる解決方法を学習しました。

#### 🧪 実装した主な処理
- **データの前処理 (`map`関数)**: 
  機械学習モデルが計算を扱えるよう、文字データ（Sex）を数値に変換。リスト要素を一括処理する効率的な手法を習得しました。
- **f-string による出力**: 
  `f"..."` を活用し、型変換を意識せずに計算結果を文字列に埋め込む、可読性の高いコード記述を実践しました。

#### 🤖 機械学習アルゴリズムとプロセス
- **DecisionTreeClassifier (決定木)**: 
  「Yes/No」の条件分岐を繰り返してデータを分類するアルゴリズムを理解しました。
- **学習と予測の分離**: 
  - **`.fit` (学習)**: 過去のデータから解き方を導き出す「インプット」の工程。
  - **`.predict` (予測)**: 学習したルールを新しいデータに適用する「アウトプット」の工程。
  この2つの明確な役割の違いを整理しました。
